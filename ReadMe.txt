
* If character falls too fast/slow, go to [play] tab, [void Init()], [world.setGravity()] and change the value of gravity.

* If character jumps too high/low,  go to [play] tab and change the value of [upForce].

* If character moves too fast/slow, go to [play] tab and change the value of [speed].

* If you meet any bug that stop you from sloving the puzzle, try to press [x] to restart this stage.

* Be careful that the boy may fall out of the right edge of the screen.

* It is easy to operate the character if you press [Up] and release it immediately. Strongly suggest not holding Up button for too long.

Enjoy your playing! :)
